### 2.1 기획 배경 - 구체화 필요? (SWOT 분석)

### 2.2 목표

     **이동성이 잦은 푸드트럭의 주소 변경을 용이하게 하며, 푸드트럭 정보를 사용자 위치 기반으로 제공함으로써
    사용자가 언제 어디서든지 주변 푸드트럭 정보를 확인할 수 있게하고, 비대면 포장 주문으로 대기 시간을 단축 → 확인(사용자 편의성 제공)
    하여 편리함을 주는 것을 목표로 서비스를 개발한다.**
